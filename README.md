# Portfolio — Yohan Jolly · BTS SIO SISR

Portfolio personnel développé en HTML/CSS/JS pur, sans dépendances externes.

## 🚀 Déploiement sur GitHub Pages

1. Crée un dépôt GitHub nommé `portfolio` (ou ton pseudo : `nahoyjolly.github.io`)
2. Upload tous les fichiers **tels quels** (en respectant la structure)
3. Va dans **Settings → Pages → Source → main branch → / (root)**
4. Ton site sera accessible à `https://ton-pseudo.github.io/portfolio/`
